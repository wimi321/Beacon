# Beacon 国际化说明

Beacon 把国际化当成产品能力，而不是发布前最后补的装饰项。

## 仓库文档语言

- English
- 简体中文

## 运行时多语言链路

1. 前端提供手动语言切换器。
2. 当前语言会持久化到本地。
3. 当前 `locale` 会透传到文字急救、视觉求助和 SOS 流程。
4. Native 侧在组装 prompt 时使用该语言，尽量保证模型输出和 UI 语言一致。
5. 阿拉伯语会启用 `dir=\"rtl\"` 和对应布局反转。

## 兜底规则

- 不能只依赖系统语言。
- 用户手动选择的语言优先级最高。
- Native 如果不能安全本地化长文本，长期建议是返回状态码，再由前端字典完成映射。
- 即使检索较弱，也应继续用真实本地推理，并保持所选语言输出。

## 多语言贡献建议

- 急救和求生文案要尽量短、直接、低歧义。
- 避免使用过强的俚语和地区黑话。
- 不要削弱危险等级和升级条件。
- 涉及方向性的 UI 变更时，要验证 RTL 布局。
- 修改 locale 相关逻辑时，请同步补测试。
