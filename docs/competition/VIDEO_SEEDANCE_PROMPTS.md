# Beacon Seedance 2.0 Prompt Library

Use this file as the copy/paste prompt bank for image-to-video generation. Shots marked `REAL PHONE PROOF` should be recorded on a real Android phone; the AI prompts are only for B-roll, transition cards, or fallback cover frames.

Global style for generated B-roll:

```text
Premium documentary survival-tech film, grounded realism, calm under pressure, cold blue-gray crisis atmosphere with subtle cyan/green Beacon glow, 16:9, cinematic lighting, shallow depth of field where appropriate, no gore, no logos, no watermark, no fake readable app UI, no exaggerated sci-fi particles.
```

## Shot 01 - No Signal

Duration: 12s.

Seed image prompt:

```text
Cinematic 16:9 close-up of a realistic hand holding a modern Android phone in a dark forest at dusk, no signal icon implied without readable UI text, rain mist, cold blue-gray atmosphere, subtle cyan glow from the phone, grounded documentary realism, shallow depth of field, no logos, no watermark, no readable text.
```

Image-to-video prompt:

```text
Slow handheld push-in toward the phone, tiny raindrops moving, distant lightning softly illuminating tree silhouettes, phone glow reflecting on fingers, tense but calm mood, cinematic realism, 24fps, no text overlays.
```

Negative prompt:

```text
fake UI text, brand logos, gore, panic expression, overdramatic lightning, cyberpunk neon, unreadable clutter, watermark
```
