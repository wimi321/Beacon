# Beacon v0.2.17 中文发布说明

Beacon `v0.2.17` 新增“战争危机”首页求生入口，并补强战争、核辐射、生物危害、化学暴露、网络攻击、停电断网和就地避险等场景的离线检索能力。

## 亮点

- 首页新增 `战争危机` 大按钮，不移除原有迷路、地震、火场、外伤、视觉求助和 SOS 流程
- 使用裁切后的透明苹果风导弹/盾牌图标，与现有首页 panic 图标保持统一
- 新增 `war_crisis` canonical scenario，确保 20 种语言都走同一底层检索意图
- 为 20 种界面语言补齐 `战争危机` 文案
- 检索层优先命中 Ready.gov 与 CDC 的爆炸、就地避险、辐射核事故、生物危害、危化暴露、网络攻击、公共场所袭击和停电断网资料
- 将 CDC 核爆/放射性应急公共指南补入离线知识库
- DFlash 只保留研究结论，不进入本次移动端生产链路
- Android 升级到 `versionCode 20` / `versionName 0.2.17`

## 电脑端验证

本轮没有安卓真机可用，因此使用电脑端验证替代真机烟测。

- 检查战争危机图标透明背景和小尺寸清晰度
- 本地 Web UI 检查首页布局、语言切换、战争危机入口和返回主页
- `npm test -- --run`
- `npm run build`
- `npm run competition:audit`
- `npm run mobile:android:release:github`
- 如本机有 Android 模拟器，则额外做模拟器烟测

## 发布产物

- `Beacon-v0.2.17-android-arm64.apk`

## 说明

APK 继续保持轻量，不内置多 GB 模型权重。用户仍在 App 内下载 Gemma 4 E2B/E4B，支持镜像 fallback 和断点续传。战争危机流程仍然是真实本地模型推理，不使用模板 fallback。
