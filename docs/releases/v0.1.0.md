# Beacon v0.1.0

Beacon `v0.1.0` is the first public GitHub release of the project.

## What this release delivers

- Real on-device emergency guidance powered by Gemma 4
- Offline-first knowledge grounding from bundled emergency and survival sources
- Session memory for continuous conversations inside the active emergency session
- Android and iOS native shells in the public repository
- Camera and local photo intake flow
- 20-language UI support with manual switching and Arabic RTL
- A polished multilingual GitHub experience with bilingual docs and contribution flows

## Included repository upgrades

- README hero demo GIF and short video
- Apache-2.0 license
- English + Simplified Chinese repository docs
- CI workflow
- Bug report and feature request forms
- PR template
- Discussion forms and community guidance

## Verification

- `npm test`
- `npm run build`
- `dart test`
- GitHub Actions CI passing on `main`

## Known limitations

- iOS release-device coverage is still expanding
- The repository excludes oversized local iOS runtime archives that exceed standard GitHub git limits
- This is a serious public pre-release, not the final app store delivery package
