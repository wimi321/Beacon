# Beacon v0.2.15 - Multilingual Release Notes

## Quick Facts

- APK: `Beacon-v0.2.15-android-arm64.apk`
- SHA256: `747e317ed66a4af4a1a05745134435b80dccd320cab1e6fda07dd8c20ccbc488`
- Size: about 34 MB
- Android: `versionCode 18` / `versionName 0.2.15`
- Offline knowledge bundle: `6,332` sources / `14,406` entries
- Model policy: the APK includes the offline knowledge base, not multi-GB Gemma weights. Download Gemma 4 E2B or E4B inside the app before offline AI use.
- Verified: `npm test`, `npm run build`, `npm run competition:audit`, `npm run competition:knowledge:safe`, `npm run mobile:android:release:github`, and APK knowledge-bundle audit.

## 简体中文

### 主要更新

- 新增 31 个许可清晰的政府/公共应急来源，覆盖 Ready.gov、CDC、NWS/NOAA、NPS。
- 离线知识库扩展到 `6,332` 个 sources 和 `14,406` 个 entries。

- 公开 APK 继续保持轻量：APK 内置知识库，不内置数 GB 级 Gemma 权重。

### 安装与使用

1. 下载并安装 `Beacon-v0.2.15-android-arm64.apk`。
2. 打开 `设置与模型`。
3. 在 App 内下载 `Gemma 4 E2B` 或 `Gemma 4 E4B`。
4. 下载完成后，即可结合本地模型和离线知识库使用 Beacon。
