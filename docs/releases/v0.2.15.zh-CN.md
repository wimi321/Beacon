# Beacon v0.2.15

Beacon `v0.2.15` 是面向 Gemma 4 Good Hackathon 的应急知识库扩容发布版，并已把新知识库打入 Android APK 发布链路。

## 亮点

- 新增 31 个许可清晰的政府/公共应急来源，覆盖 Ready.gov、CDC、NWS/NOAA、NPS
- 扩展离线覆盖范围：龙卷风、飓风、冬季天气、海啸、山火、家庭化学品事故、安全饮水、食品安全、停电、灾后清理、霉菌清理、徒步与营火安全
- 离线知识库更新到 `6,332` 条 source 记录和 `14,406` 条 entries
- 保守的比赛安全知识预览扩展到 `126` 个 sources 和 `1,477` 个 entries
- 新增安全饮水、龙卷风、化学品泄漏场景的检索回归测试
- 公开 APK 继续保持轻量：Gemma 4 E2B/E4B 权重仍由用户在 App 内下载，离线知识库随 APK 内置
- Android 版本升级到 `versionCode 18` / `versionName 0.2.15`

## 验证

- `npm test`
- `npm run build`
- `npm run competition:audit`
- `npm run competition:knowledge:safe`
- `npm run mobile:android:release:github`

## APK 安装流程

1. 从 GitHub Releases 下载并安装 APK。
2. 打开 `设置与模型`。
3. 在 App 内下载 `Gemma 4 E2B` 或 `Gemma 4 E4B`。
4. 下载完成后即可离线使用 Beacon，APK 已内置扩容后的应急知识库。

## Release 资产

- `Beacon-v0.2.15-android-arm64.apk`

## 说明

APK 内置的是扩容后的离线知识库，不内置数 GB 级 Gemma 模型权重。这样可以让 GitHub 安装包保持可下载、可分发，同时在用户完成模型下载后继续保证真实端侧推理。
