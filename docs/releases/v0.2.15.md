# Beacon v0.2.15

Beacon `v0.2.15` packages the expanded emergency knowledge base into the Android APK release path for the Gemma 4 Good Hackathon push.

## Highlights

- Adds 31 additional government/public emergency sources from Ready.gov, CDC, NWS/NOAA, and NPS
- Expands coverage for tornadoes, hurricanes, winter weather, tsunamis, wildfires, household chemical incidents, safe water, food safety, power outages, disaster cleanup, mold cleanup, and hiking/campfire safety
- Updates the offline knowledge bundle to `6,332` source records and `14,406` entries
- Expands the conservative competition-safe preview to `126` sources and `1,477` entries
- Adds retrieval tests for safe water, tornado, and chemical spill scenarios
- Keeps the public APK lightweight: users still download Gemma 4 E2B/E4B in-app, while the offline knowledge base is bundled in the APK
- Bumps Android to `versionCode 18` / `versionName 0.2.15`

## Verification

- `npm test`
- `npm run build`
- `npm run competition:audit`
- `npm run competition:knowledge:safe`
- `npm run mobile:android:release:github`

## APK Install Flow

1. Install the APK from GitHub Releases.
2. Open `Settings & Models`.
3. Download `Gemma 4 E2B` or `Gemma 4 E4B` in-app.
4. Use Beacon offline with the bundled emergency knowledge base.

## Release Asset

- `Beacon-v0.2.15-android-arm64.apk`

## Notes

The APK includes the expanded offline knowledge bundle, but not multi-GB Gemma model weights. This keeps the release installable from GitHub while preserving real on-device inference after the user downloads a local model.
