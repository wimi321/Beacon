# Beacon v0.2.17 - Multilingual Release Notes

APK: `Beacon-v0.2.17-android-arm64.apk`
Android: `versionCode 20` / `versionName 0.2.17`

## English

Adds a War Crisis shortcut with a matching transparent Apple-style icon and stronger offline grounding for civilian conflict, radiation, biohazard, chemical, cyberattack, outage, and shelter-in-place scenarios.

## 简体中文

新增“战争危机”入口和透明苹果风图标，并强化战争、核辐射、生物危害、化学暴露、网络攻击、停电断网与就地避险场景的离线知识检索。

## 繁體中文

新增「戰爭危機」入口和透明蘋果風圖示，並強化戰爭、核輻射、生物危害、化學暴露、網路攻擊、停電斷網與就地避險場景的離線知識檢索。

## 日本語
