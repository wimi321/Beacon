# Beacon v0.2.16 - Multilingual Release Notes

APK: `Beacon-v0.2.16-android-arm64.apk`  
Android: `versionCode 19` / `versionName 0.2.16`

## English

This release makes visual rescue answers stream like normal emergency chat, keeps captured photos visible in the conversation, and changes the SOS button copy to tap-only wording.

## 简体中文

本版本让视觉求助像普通急救对话一样流式输出，拍照图片会保留在对话框中，底部 SOS 文案也统一改成点击语义。

## 繁體中文

本版本讓視覺求助像一般急救對話一樣流式輸出，拍攝圖片會保留在對話框中，底部 SOS 文案也統一改為點擊語義。

## 日本語

このリリースでは、視覚支援の回答も通常の緊急チャットと同じようにストリーミング表示され、撮影画像が会話内に残り、SOS ボタンの文言がタップ操作に統一されました。

## Español
