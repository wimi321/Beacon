# Beacon v0.1.1

Beacon `v0.1.1` is the first release-candidate style hardening pass after the public repository launch.

## What changed in this release

- Stabilized the install-first delivery flow for Android release artifacts
- Kept GitHub distribution lightweight while preserving real on-device Gemma 4 behavior
- Hardened mobile startup, model onboarding, and release build reliability across Android and iOS

## Key product fixes

- Fixed the first-launch and cold-launch home-state bug where system-only warning messages could incorrectly push the app into chat view
- Preserved the simple emergency-first home screen when only low-battery guidance is present
- Kept model onboarding explicit through `Settings & Models` without faking AI availability before the local model is ready

## Android release hardening

- Reduced Android lint noise by cleaning template leftovers and normalizing launcher/splash assets
- Moved to an AGP/Kotlin combination compatible with current R8 metadata parsing expectations
- Reworked `mobile:build` so `cap sync` is followed by an automatic Android generated-build normalization pass
- Revalidated debug APK, release APK, and release AAB output generation

## iOS validation updates

- Revalidated Debug simulator and Release device builds with Xcode
- Reinstalled and cold-launched the app in the iOS simulator
- Confirmed that the cold-launch path returns to the emergency home screen instead of an empty chat shell

## Verification run for this release

- `npm test`
- `npm run build`
- `npm run mobile:build`
- `dart analyze`
- `dart test`
- `cd android && ./gradlew :app:testDebugUnitTest :app:testReleaseUnitTest :app:lintDebug :app:assembleDebug :app:assembleRelease :app:bundleRelease`
- `xcodebuild -project ios/App/App.xcodeproj -scheme App -configuration Debug -sdk iphonesimulator -destination 'generic/platform=iOS Simulator' CODE_SIGNING_ALLOWED=NO build`
- `xcodebuild -project ios/App/App.xcodeproj -scheme App -configuration Release -sdk iphoneos -destination 'generic/platform=iOS' CODE_SIGNING_ALLOWED=NO build`
- `xcrun simctl install <simulator> App.app`
- `xcrun simctl launch <simulator> com.beacon.sos`
